# Control de Velocidad: Asistencia de Carril de Boxes (PLA) - README de Configuración

Este documento detalla los pasos de configuración y las indicaciones necesarias para la puesta en marcha de la simulación del sistema de control de velocidad del tipo Pit Lane Assistance.

---

## Requisitos Obligatorios

Para la correcta ejecución del script en tu entorno, es indispensable contar con lo siguiente:

### 1. Entorno de Desarrollo y Versión de Python

| Requisito | Detalle |
| :--- | :--- |
| **Intérprete Python** | **Descargar e instalar Python 3.12 o superior.** |
| **IDE** | **Visual Studio Code (VS Code)** es el entorno recomendado para la ejecución. |

### 2. Instalación de Dependencias

El script requiere las bibliotecas `numpy` (para operaciones numéricas) y `matplotlib` (para la visualización en tiempo real).

Abre el terminal en el directorio del proyecto y ejecuta el siguiente comando para instalarlas:

```bash

pip install numpy matplotlib

## Puesta en Marcha (Ejecución de la Simulación)

La consigna requiere que este archivo contenga las indicaciones necesarias para la ejecución. Sigue estos pasos para iniciar la simulación:

1.  **Abrir el Proyecto:** Abre la carpeta que contiene el archivo de código Python en **Visual Studio Code**.
2.  **Abrir el Archivo:** Abre el archivo principal de la simulación (`.py`) en el editor.
3.  **Ejecutar el Script:**
    * Asegúrate de que tu terminal esté en el directorio correcto.
    * Haz clic derecho en el editor y selecciona **"Run Python File in Terminal"** o utiliza el botón de **"Run" (▶️)** en la interfaz de VS Code.

La simulación se iniciará en una nueva ventana con los gráficos en tiempo real, y en la terminal de VSCode se podran ver los logs actuales de la simulación.
